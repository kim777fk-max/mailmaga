# メルマガいたしん 統合管理ツール

板診会のメールマガジン（メルマガいたしん）制作フローを効率化するためのローカル統合管理ツールです。

## 構成

```
mailmaga/
├── app.py              # Flaskアプリ本体（統合管理ツール）
├── requirements.txt
├── start.sh            # 起動スクリプト
├── templates/          # HTMLテンプレート
├── static/             # CSS
├── data/               # 実行時データ（.gitignore対象）
│   └── cycles.json     # 号ごとの管理データ
└── form/
    └── melmaga.html    # 原稿入力フォーム（XServerに設置）
```

## セットアップ

```bash
pip3 install flask
```

## 起動方法

```bash
./start.sh
# または
python3 app.py
```

ブラウザで http://localhost:5001 を開く。

| ⑦ | フォーマット転記 | 広報部 |

| ⑨ | メルマガ配信 | 広報部 |
